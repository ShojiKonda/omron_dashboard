OMRON Classroom Data Lab

Activity Rhythm Dashboard

OMRON活動量計のsummary CSVとprocessed CSVを読み込み、歩数、エクササイズ量、1日のMETs時系列、個人平均と全体平均の違いを可視化します。

Browser only No login No upload to server
学生データはブラウザ内で処理 CSVは外部サーバーへ送信されません
1

データを読み込む

summary CSVを1つ、processed CSVを複数選択します。ファイルはブラウザ内で処理されます。

解析対象日数 0日 データ未読込
平均歩数 - 装着180分以上の日で算出
可視化対象日数 0日 装着時間180分以上
平均METs - processedから算出
平均エクササイズ - 装着180分以上の日で算出
2

サマリーデータの可視化

装着時間が180分以上の日だけを対象に、歩数とエクササイズ合計を日別棒グラフで確認します。点線は全員平均です。

3

各個人の各日の時系列データ

processed CSVから、選択日のMETs時系列を表示します。「全日」を選ぶと全ファイルを重ね描きします。

個人データ 縦軸: 表示範囲に合わせて自動調整
4

個人平均と全体平均の比較

各個人のprocessed CSVから計算した平均パターンと、全平日平均を比較します。横軸の表示範囲を選択できます。

平日平均METs: 月〜金

月曜〜金曜の全員平均METsを比較します。横軸の表示範囲を選択できます。
