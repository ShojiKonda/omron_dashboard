OMRON Classroom Data Lab

Activity Rhythm Dashboard

OMRON活動量計のsummary CSVとprocessed CSVを読み込み、装着時間、歩数、エクササイズ量、1日のMETs時系列、個人平均と全体平均の違いを可視化します。

Browser only No login No upload to server
学生データはブラウザ内で処理 CSVは外部サーバーへ送信されません

対応するCSV形式
  • summary: OMRON出力形式、または date, weekday, steps, wear_minutes, exercise_ex などの簡易形式
  • processed: minute, time, METs, flag の4列形式。ヘッダーなしでも可
  • 文字コード: UTF-8 / Shift-JIS / CP932 を自動判定します
