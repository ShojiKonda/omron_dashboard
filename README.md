# OMRON Activity Dashboard

授業用の活動量データ可視化ダッシュボードです。学生がブラウザ上でOMRON由来のCSVデータを読み込み、装着時間、歩数、METs時系列、曜日差、全体平均との比較を確認できます。

## 目的

これまでExcelで行っていた以下の作業をWebブラウザ上で実行します。

1. summaryファイルで装着時間を確認する
2. summaryファイルで歩数を確認する
3. 各日のprocessedファイルから個人平均を作成し、全体平均と比較する
4. 曜日間比較を行う
5. 結果をもとに考察する

## 使い方

1. GitHub Pagesで公開されたURLを開きます。
2. `demo_20221021.csv` のようなsummaryファイルをアップロードします。
3. `demo_20221021_processed_20221021.csv` のようなprocessedファイルを複数選択します。
4. 必要に応じて、全体平均ファイルをアップロードします。
5. ダッシュボード上で装着時間、歩数、METs時系列、曜日比較、ヒートマップ、考察サポートを確認します。

## データの扱い

このアプリは静的Webアプリです。アップロードしたCSVはブラウザ内でのみ処理され、サーバーには送信されません。ログインもデータベースも使用しません。

## 対応データ

### Summary CSV

日本語版OMRON出力に対応しています。主な列として以下を探します。

- 日付
- 曜日
- 歩数合計値
- 装着時間
- カロリー
- エクササイズ量

### Processed CSV

1分ごとの時系列データを想定しています。

```csv
minute,time,mets,flag
0,00:00:00,0.0,-1.0
1,00:01:00,0.0,-1.0
...
```

ヘッダーなしの次の形式にも対応します。

```csv
0,00:00:00,0.0,-1.0
1,00:01:00,0.0,-1.0
...
```

## GitHub Pages公開

このリポジトリにはGitHub Pages用のworkflowを含めています。

GitHubのリポジトリ画面で以下を確認してください。

1. Settings
2. Pages
3. Build and deployment
4. Source: GitHub Actions

その後、`main` ブランチにpushされると自動的に公開されます。
